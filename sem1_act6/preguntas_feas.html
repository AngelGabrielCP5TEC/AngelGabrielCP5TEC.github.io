HTML

Introduction

What is HTML?

Es un hiper texto usado para programar páginas web de forma más sencilla y versatil

Explain the head and the body tags.

La etiqueta head contiene cierta información específica del documento como el título, enlaces y otra información que el usuario no ve. Mientras que etiqueta body contiene el contenido visible para el usuario de la página web como texto, imágenes, videos, etc.

Basic Tags

Explain the section, article, aside, header, footer tag.

  • section: Define una sección en un documento. Puede ser usado para agrupar contenido temáticamente relacionado.
  • article: Representa un contenido autónomo y auto-contenido, como un artículo de un blog o una noticia.
  • aside: Define un contenido aparte del contenido principal, como una barra lateral o una nota al margen.
  • header: Representa un grupo de ayudas introductorias o de navegación. Puede contener títulos, logotipos, o menús de navegación.
  • footer: Define el pie de página de un documento o sección, usualmente contiene información de autoría, derechos de autor, o enlaces a términos de uso.

CSS

Introduction

What is CSS?

CSS es un lenguaje de programacion enfocado al diseño para páginas web.

What is the purpose of CSS?

Separar el contenido de la presentación en una página web. Esto permite definir estilos visuales, como colores, fuentes, espacios, y posiciones de los elementos HTML.

How do you apply a CSS to an HTML (explain the 3 options).

  • Usando el atributo style directamente en la etiqueta HTML.
  • Usando la etiqueta style dentro del head.
  • Usando un archivo CSS externo vinculado con un link en el head.

What is a selector? What is a CSS rule?

Un selector es un patrón que coincide con los elementos a los que se le quiere dar estilo. Una regla CSS es un conjunto de propiedades, valores o atributos aplicados a los elementos seleccionados.

Basic CSS

Explain a:

  • Tag Selector: Selecciona todos los elementos de un tipo especificado. Ejemplo: p { color: red; } selecciona todos los párrafos.
  • Class Selector: Selecciona todos los elementos con una clase especificada. Ejemplo: .miClase { color: red; } selecciona todos los elementos con la clase "miClase".
  • Tag and Class Selector: Combina un selector de etiqueta y una clase. Ejemplo: p.miClase { color: red; } selecciona los párrafos con la clase "miClase".
  • Id Selector: Selecciona un elemento con un id especificado. Ejemplo: #miId { color: red; } selecciona el elemento con el id "miId".

What is a class? What is an id? How are they related to CSS?

Una clase es un atributo que define un conjunto de estilos noo tamando en cuenta la etiqueta a la que será aplicada. Un id es un atributo que identifica de manera única un solo elemento. En CSS, las clases se definen con un punto (.) y los ids con un simbolo de gato (#).

Explain the CSS Box Model? Show an example.

El modelo de caja CSS es un concepto que describe el diseño y el espacio de cada elemento en la página web. Incluye márgenes, bordes, rellenos y el contenido real del elemento. Por ejemplo:

div{
width: 100px;
height: 100px;
padding: 10px;
border: 5px solid black;
margin: 20px;
}
