Buscador de peliculas con API

Para este trabajo se hizo uso de una API conectada a la base de datos de IMDb y al framework de css de sakura. En esta página el usuario puede ingresar el nombre de la pelicula que desea buscar y mediante el endpoint de autocompletar se regresara una lista de peliculas que cumplan con ese nombre o muy similar a ese nombre, y esta lista será mostrada al usuario.
